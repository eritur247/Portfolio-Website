# Portfolio Website

Personal portfolio showcasing AI engineering, data science, and full-stack development projects.

## Tech Stack

- **Frontend**: React, JavaScript
- **Build Tool**: Vite
- **Styling**: Custom CSS with fluid animations
- **Deployment**: Vercel

## Features

- Interactive fluid animation background
- Responsive design
- Project showcase
- Work experience timeline
- Contact integration

## Development

```bash
npm install
npm run dev
```

## Deployment

Automatically deployed to Vercel on push to main branch.
